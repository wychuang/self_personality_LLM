# self_personality_LLM 实施计划

## 攻击路径: Exp4 → Exp1 → Exp2 → Exp3

### 第1周: 实验4 —— 人格固化曲线（纯观测）
- Pythia 143 checkpoint 迭代器
- 行为版 Big Five 探针 + 风格指纹 + 价值观选择
- 固化曲线图 + 关键期检测

### 第2周: 实验1 —— 身份方向锚定 (IDA)
- 从种子叙事提取 v_id（残差流平均激活）
- 双损失训练: L = CE + λ * L_ID
- λ 扫描 [0.01, 0.05, 0.1, 0.2, 0.5, 1.0]
- 偏置注入变体 + 对抗性评估

### 第3-5周: 实验2 —— 自我一致性RL
- 500 开放问题，两次回答间隔10K token
- 深层立场一致性评分 → DPO 训练
- 对抗性前缀淬火

### 第6-8周: 实验3 —— 驱动力嵌入训练
- 好奇心驱动 + 自我连贯驱动 + empowerment 驱动
- 漏积分器内部状态 + 自我叙事记忆
- 多目标联合训练: L = L_LM + α*L_curi + β*L_coh + γ*L_emp

## MVP 策略
- Pythia-160M 快速迭代 → Pythia-1.4B/6.9B → LLaMA-7B/13B
- 所有实验用 LoRA (rank=8~64) 冻结基础模型
- OmegaConf YAML 配置，零硬编码
- W&B 全量日志
